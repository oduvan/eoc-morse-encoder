Ваша задача - зашифровать текстовое сообщение используя азбуку Морзе. Исходный текст будет состоять только из букв (в верхнем и нижнем регистре), чисел и пробелов. Специальные символы использоваться не будут ('&', '?', '#' и т.п.)
Используйте 1 пробел между буквами одого слова и 3 пробела между отдельными словами.

Входные данные: Секретное сообщение как строка

Выходные данные: Зашифрованный текст

Примеры:

{% if interpreter.slug == "js-node" %} 
morseEncoder("some text") == "... --- -- .   - . -..- -"
morseEncoder("2018") == "..--- ----- .---- ---.."
morseEncoder("It was a good day") == ".. -   .-- .- ...   .-   --. --- --- -..   -.. .- -.--"
{% else %} 
morse_encoder("some text") == "... --- -- .   - . -..- -"
morse_encoder("2018") == "..--- ----- .---- ---.."
morse_encoder("It was a good day") == ".. -   .-- .- ...   .-   --. --- --- -..   -.. .- -.--"
{% endif %}

Как это используется: Для криптографии, передачи и защиты важной информации.

Предусловия:
0 < len(text) < 50
Текст будет содержать только буквы английского алфавита (в верхнем и нижнем регистре), числа и пробелы.
